# Peony 注重隐私的担保交易系统

> PS：这是一个学习练手项目，请勿用于违法途径，否则后果自负。

idea：见过太多人在抱怨私下交易被骗的情况，那么是不是可以拥有一个平台做担保以保证交易的正常进行？

### 技术栈
- go1.20.2
- gin v1.9.1
- go-redis v9.0.5
- viper v1.16.0
- goPay v1.5.95
- gorm v1.25.2
- mysql 8.0.31
